# MPEP Routing

## Purpose

Use this file to route questions about United States patent examination, PCT, Hague international design applications, and MPEP topics to the local Markdown cache generated from the MPEP E9R-01-2024 PDFs.

## Cache Location

- Bundled Markdown cache: `references/mpep-e9r-01-2024/`
- Cache index: `references/mpep-e9r-01-2024/INDEX.md`
- Original source: USPTO MPEP E9R-01-2024 PDF set

## Routing Rules

- Start with `references/mpep-e9r-01-2024/INDEX.md`.
- If the correct chapter is unclear, search `mpep-0000-table-of-contents.md` or `mpep-9090-subject-matter-index.md`.
- Read only the specific chapter file needed for the question.
- Cite the MPEP version as E9R-01-2024 when summarizing rule-based content.
- Treat the cache as reference material, not legal advice or a guarantee of USPTO outcome.
