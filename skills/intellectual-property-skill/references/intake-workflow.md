# Intake Workflow

## Current Status

This file defines the shared intake structure. Fill each business module with company-approved details later.

## Universal Intake Fields

- User's main goal
- Business type: trademark, patent, copyright, declaration, mixed IP, or uncertain
- Jurisdiction or target country/region
- Applicant type: individual, company, institution, or other
- Current stage: early consultation, preparing materials, already filed, received notice, renewal/maintenance, dispute/risk
- Urgency or deadline
- Existing materials
- Questions the user wants answered

## Trademark Intake

- Mark text, logo, or combined mark
- Goods or services involved
- Current brand usage status
- Target classes if known
- Whether a search has been done
- Filing country/region
- Applicant identity type
- Any prior refusal, objection, opposition, or similar risk

## Patent Intake

- Patent type: invention, utility model, design, or uncertain
- Technical field or product category
- Whether the idea has been publicly disclosed
- Whether drawings, specification, claims, or prototype materials exist
- Filing country/region
- Priority or deadline concerns
- Current stage: search, drafting, filing, examination, payment, maintenance, transfer

## Copyright Intake

- Work type: software, text, image, video, music, design, database, or other
- Author or owner
- Completion or publication date
- Whether source files or proof of creation exist
- Registration country/region
- Purpose: evidence, platform requirement, licensing, financing, dispute, other

## Declaration Intake

- Declaration type or target authority
- Related IP matter
- Declarant identity
- Required language or notarization/legalization need
- Deadline
- Existing official notice or template

## Handoff Readiness

Offer a Markdown handoff brief when at least these facts are known:

- Business type
- Jurisdiction
- Applicant type
- User's main goal
- Current stage
- Key materials already available
- Key missing materials or open questions
