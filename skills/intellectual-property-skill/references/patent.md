# Patent Reference

## Scope

Use this file for invention, utility model, design patent, filing preparation, technical disclosure, priority, examination, grant, payment, maintenance, patent search, and novelty-risk orientation. For customer-facing wording, apply `response-style.md` first.

## Pricing And MPEP References

- For patent prices, timelines, and required materials, read `references/pricing/ip-service-pricing-2026-05-16.md`. Do not mention snapshot dates, source files, or internal notes in customer-facing answers.
- For US patent examination, PCT, Hague, patentability, eligibility, novelty, obviousness, or USPTO procedure questions, read `references/mpep-routing.md`.

## Minimum Response Rule

If the requested jurisdiction or patent type is not covered in the references, answer briefly with general orientation and say: `这个专利类型/地区的具体材料和费用我建议再确认一下。` Then ask whether the user wants a handoff brief.
