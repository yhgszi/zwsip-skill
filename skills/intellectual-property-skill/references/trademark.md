# Trademark Reference

## Scope

Use this file for trademark registration, classification, required materials, office action, refusal, opposition, renewal, transfer, licensing, and change-service questions. For customer-facing wording, apply `response-style.md` first.

## Pricing Reference

For trademark prices, timelines, and required materials, read `references/pricing/ip-service-pricing-2026-05-16.md`. Do not mention snapshot dates, source files, or internal notes in customer-facing answers.

## Minimum Response Rule

If the requested jurisdiction or service is not covered in the references, answer briefly with general orientation and say: `这个地区/事项的具体材料和费用我建议再确认一下。` Then ask whether the user wants a handoff brief.
